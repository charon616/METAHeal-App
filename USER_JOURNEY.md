# User Journey

- patient go to see physical therapy doctor
- doctor create training menus, and send VR training kits to patients
- user open smartphone app, and confirm their menu today
- put the VR, and training with fun game-like experience
- the system evaluate the user movement precision automatically
- after all trainings, user get report of that day, percentage of precision and their history and improvements
- the results automatically send to smartphone apps, and apps automatically send that data to allocated doctor
- doctor check the patient training process through smartphone, and do remote advice if needed.
- user can keep training easily, and release from the stress of booking offline session
